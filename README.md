# Pasos para ejecutar la página

Sigue estos comandos para instalar las dependencias necesarias y levantar el proyecto correctamente.

## 1. Instalar todas las dependencias del proyecto
```bash
npm install
```

Este comando instala todas las dependencias definidas en el archivo `package.json`.

## 2. Instalar React Router DOM
```bash
npm install react-router-dom
```

Este paquete permite manejar las rutas y la navegación dentro de la aplicación React.

## 3. Instalar SweetAlert2
```bash
npm install sweetalert2
```

Este paquete permite mostrar alertas, mensajes y ventanas emergentes más bonitas e interactivas.

## Resumen
Después de ejecutar los comandos anteriores, el proyecto quedará listo para funcionar correctamente con sus rutas y alertas.
